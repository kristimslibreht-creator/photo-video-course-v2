# Kristina Course Site (Next.js) — MVP

Это готовый минимальный сайт:
- Лендинг `/`
- Страница оплаты `/buy`
- Вход по паролю `/login`
- Закрытый раздел уроков `/course` и `/course/[slug]`

## 1) Запуск локально
```bash
npm i
npm run dev
```
Открой: http://localhost:3000

## 2) Установка пароля (важно)
Создай файл `.env.local` в корне и добавь:
```
COURSE_PASSWORD=ТВОЙ_ПАРОЛЬ_ДЛЯ_УЧЕНИКОВ
```

## 3) Добавить/изменить уроки и видео
Открой `lib/course.ts`:
- `title`
- `duration`
- `videoEmbedUrl` (например: https://www.youtube.com/embed/VIDEO_ID)
- `summary`
- `tasks`

## 4) Вставить Kaspi QR/текст оплаты
Открой `app/buy/page.tsx` и замени блок справа на свою картинку/текст.

## 5) Деплой на Vercel (самый простой)
1. Создай репозиторий на GitHub и залей туда проект
2. Зайди на vercel.com → New Project → выбери репозиторий
3. В Settings → Environment Variables добавь:
   - COURSE_PASSWORD = твой пароль
4. Deploy

## Примечание по оплатам
Этот MVP сделан под быстрый запуск (Kaspi/перевод + выдача пароля вручную).
Когда будут продажи, можно добавить:
- авто-выдачу доступа после оплаты (webhooks)
- личные кабинеты
- отдельные пароли/аккаунты для учеников
